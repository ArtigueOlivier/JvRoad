Création d'un road book: Modifier le copyright

Choisir Copyright dans le menu Données, une boîte de dialogue apparaît pour vous permettre de saisir le copyright du road book. Dans le cadre de cette formation, nous entrons "Bourbier Corporation".



Un click sur le bouton "Ok" permet de valider la modification.
