JvRoad Manuel de référence

La boîte de dialogue: Livre - Options d'affichage







Description générale

Cette boîte de dialogue permet de renseigner les différentes caractéristiques d'un road book au travers d'un ensemble de champs répartis sur deux onglets.

Les champs de l'onglet Identification

Le champ "Titre":
Pour indiquer le titre du road book.

Le champ "Copyright":
Pour indiquer le copyright du road book.

Le champ "Numéro des cases":
Pour indiquer si le numéro des cases doit être affiché ou pas.

Le champ "Sens de la lecture":
Pour indiquer le sens de lecture du road book.

Le champ "Cases par colonnes":
Pour indiquer le nombre de cases dans chaque colonne du road book.

Le champ "Taille de la police":
Pour indiquer la taille de police utiliser pour le texte du commentaire et de la carte texte.

Le champ "Justification texte"
Pour indiquer le type de justification (Basique ou Evoluée) au niveau du texte. Sur certaines machines Java, la justification évoluée ne fonctionne pas correctement (avec suppression des espaces par exemple lors de l'affichage), le mode Basique (moins esthétique) fonctionne dans tous les cas.

L'onglet Statistiques

Cet onglet affiche un ensemble de statistiques concernant le road book.

L'onglet Filtres

Cet onglet permet de gérer les filtres.
Les filtres permettent de modifier la présentation et le dessin sur une page lors de l'impression. Ainsi, vous pouvez utiliser un filtre et un seul par page pour modifier l'autre d'affichage des cases. Les ordres disponibles sont:
  		• Identique au sens actuel.
  		• Inversé.
  		• Mélangé.
    Au niveau du dessin, vous pouvez indiquer que sur une page, toutes les routes et les routes nationales seront finalement dessinées comme des chemins.

    Les boutons
    Permet d'ajouter un filtre.
    Permet de supprimer tous les filtres qui sont cochés.

    Note: Les effets des filtres ne sont visibles que lors de l'impression ou dans la fenêtre "Aperçu avant impression".
