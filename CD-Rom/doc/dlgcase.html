La boîte de dialogue: Case





Description générale

Cette boîte de dialogue permet de renseigner les différentes caractéristiques de la case au travers d'un ensemble de champs répartis sur deux onglets.

Les champs de l'onglet Général

Le champ "Partiel":
Contient la distance partiel de la case. L'unité de distance est indiquée à droite de la zone de saisie.

Le champ "Remise à zéro":
Pour indiquer si une remise à doit être fait au niveau de cette case.

Le champ "Distance approximative":
Pour indiquer si la distance indiquée est approximative au niveau de cette case.

Le champ "Distance inconnue":
Pour indiquer que cette case ne possède pas de distance.

Le champ "Indication numéro 1":
Pour indiquer la première indication qui doit figurer sous la forme de petit dessin dans la case.

Le champ "Indication numéro 2":
Pour indiquer la deuxième indication qui doit figurer sous la forme de petit dessin dans la case.

Le champ "Indication numéro 3":
Pour indiquer la troisième indication qui doit figurer sous la forme de petit dessin dans la case.

Le champ "Indication numéro 4":
Pour indiquer la quatrième indication qui doit figurer sous la forme de petit dessin dans la case.

Les champs de l'onglet Texte

Le champ "Texte centré":
Si ce champ est coché alors le texte sera affiché centré dans l'autre cas, il est aligné à gauche.

Le champ "Zone de saisie de texte":
Cette zone est utilisée pour indiquer le texte associé à la case. La longueur maximale du texte dépend des paramètres de présentation.
