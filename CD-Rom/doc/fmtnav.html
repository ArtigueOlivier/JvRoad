Création d'un road book: Utiliser le navigateur

 Cette fenêtre affiche un aperçu du road book sous la forme d'un long film. Par ailleurs, cette fenêtre possède un ensemble interactions:

Click droit
Sur l'une des cases: Affiche un menu à la position de la souris qui permet de:
- Copier le contenu de la case dans le presse papier.
- Copier les cartes de la case dans le presse papier.
- Copier la courante de la case dans le presse papier.
- Positionner le navigateur sur la première case.
- Positionner le navigateur sur la case précédente.
- Positionner le navigateur sur la case suivante.
- Positionner le navigateur sur la dernière case.

Double click
Affiche la case située sous le double click dans la fenêtre d'édition.
