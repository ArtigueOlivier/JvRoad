JvRoad Manuel de référence

La boîte de dialogue: Rechercher - Résumé



Description générale

Cette boîte de dialogue permet de visualiser le road book sous forme condensée et de se positionner sur une case. Chaque case est résumée sur une ligne du tableau.

Les interactions

Click gauche:
Un click gauche sur une ligne du tableau, sélectionne la dite ligne.

Les boutons

Le bouton "Aller":
Pour positionner le road book sur la case correpondante à la ligne sélectionnée. Une fois le road book positionné, cette boîte de dialogue se ferme automatiquement.

Le bouton "Fermer":
Pour fermer la boîte de dialogue sans changer la position courante du road book.
