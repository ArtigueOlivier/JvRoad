La boîte de dialogue: Enregistrement



Description générale

Cette boîte de dialogue permet de renseigner le nom du fichier qui sera utilisé pour sauvegarder le contenu du road book. L'aspect de cette boîte de dialogue peut varier en fonction du système de la machine et de la langue. La version présentée ici est celle de Windows XP en langue française.
