JvRoad Manuel de démarrage

Installation

Prérequis
Pour fonctionner JvRoad a besoin d'une machine Java. A la rubrique télécharger du site Road.fr, vous avez la possibilité de télécharger la machine Java pour Windows version 32 bits (NT, 2000 ou XP). Pour les autres plateformes (système) vous pouvez aller sur le site de Sun Microsystem.

Windows
Installer JvRoad à parti d'un CD-Rom
  		•  Insérer le CD-Rom dans le lecteur.
  		• Exécuter le programme setup.exe situé sur le CD-Rom.
  		• Suivre les instructions.

    Installation par téléchargement
  		•  Télécharger JvRoad pour Window.
  		• Enregistrer le fichier zip dans un répertoire sur votre disque dur.
  		• Extraire le contenu du fichier zip dans ce même répertoire.
  		• Exécuter le programme setup.exe.
  		• Suivre les instructions.

    Linux et autres plateformes supportant Java
    Installer JvRoad à parti d'un CD-Rom
  		•  Insérer le CD-Rom dans le lecteur.
  		• Créer un répertoire JvRoad sur votre disque dur.
  		• Copier le contenu du CD-Rom dans le répertoire JvRoad.

    Installation par téléchargement
  		•  Télécharger JvRoad pour Linux.
  		• Créer un répertoire JvRoad sur votre disque dur.
  		• Enregistrer le fichier JvRoad.tar.gz dans le répertoire JvRoad.
  		• Extraire le contenu du fichier JvRoad.tar.gz dans ce même répertoire. Sous Linux l'opération s'effectue en deux temps, d'abord une décompression avec gunzip puis un tar xvf JvRoad.tar.

    Saisir la licence pour les versions Light ou Pro
    Cette partie de l'installation concerne uniquement les acheteurs d'une version Light ou Pro.
  		• Lancer JvRoad.
  		• Choisir Licence dans le Aide.
  		• Saisir votre licence.
  		• Click sur le bouton Ok. Un message vous indique si la licence est valide. Si la licence est indiquée comme "invalide", nous vous conseillons de vérifier que vous n'avez pas commis d'erreur de saisie.
  		•  Quitter puis relancer JvRoad.

    Activer le mode Reader
    Le mode reader permet de configurer JvRoad de manière à ce que les utilisateurs puissent uniquement lire le road book, aucune modification n'est possible.
  		• Lancer JvRoad.
  		• Choisir Licence dans le Aide.
  		• Saisir comme licence, la clé suivante: READER-JVROAD-READER-JVROAD-READER-JVROAD.
  		• Click sur le bouton Ok. Un message vous indique si la licence est valide. Si la licence est indiquée comme "invalide", nous vous conseillons de vérifier que vous n'avez pas commis d'erreur de saisie.
  		•  Quitter puis relancer JvRoad.
