JvRoad

Manuel de référence

Ce manuel se décompose en quatre parties:

Les commandes du menu:
Cette partie est une description de toutes les commandes du menu y compris celles qui ne sont pas accessibles dans la version d'évaluation.

Les boutons:
Cette partie est une description de toutes les commandes associées aux différents boutons y compris ceux qui ne sont pas accessibles dans la version d'évaluation.

Les interactions:
Cette partie est une description de toutes les interactions possibles aux niveaux des différentes fenêtres. Certaines interactions ne sont accessibles que dans la version professionnelle.

Les boîtes de dialogues:
Cette partie est discription des différentes boîtes de dialogues y compris celles qui ne sont pas accessibles dans la version d'évaluation.
