JvRoad Manuel de référence

La boîte de dialogue: Edition langues



Description générale

Cette boîte de dialogue permet de créer ou d'éditer un dictionnaire. Chaque dictionnaire contient l'ensemble des termes textuels utilisés par le logiciel. Grâce à cette boîte de dialogue, vous pouvez traduire le logiciel dans la langue de votre choix à condition qu'elle utilise l'alphabet latin.

Les boutons

:
Pour créer un nouveau dictionnaire.

:
Pour ouvrir un dictionnaire existant.

:
Pour enregistrer les modifications effectuées.

:
Pour enregistrer les modifications effectuées dans un nouveau dictionnaire.

Valider:
Pour valider la modification d'un texte.

Les champs

Le champ "Fichier":
Contient le nom du fichier correspondant au dictionnaire.

Le champ "Remplacer":
Contient le texte que vous souhaitez modifier.

Le champ "Par":
Contient le texte qui remplace dans le dictionnaire.

Les interactions

Double click:
Un double click sur une des lignes du dictionnaire permet de sélectionner cette entrée afin de pouvoir l'éditer dans le champ "Par".
