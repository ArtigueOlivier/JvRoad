JvRoad Manuel de référence

Les commandes du menu Données

Distance   (, Alt+DD ou Ctrl+K)
Pour éditer la distance. Une boîte de dialogue apparaît pour saisir les informations.

Commentaires   (, Alt+DC ou Ctrl+M)
Pour éditer le texte de la case. Une boîte de dialogue apparaît pour saisir les informations.

Indications   (, Alt+DI ou Ctrl+Q)
Pour éditer les indications. Une boîte de dialogue apparaît pour saisir les informations.

Titre   (, Alt+DT ou Ctrl+T)
Pour éditer le titre du road book. Une boîte de dialogue apparaît pour modifier le titre.

Copyright   (, Alt+DY ou Ctrl+H)
Pour éditer le copyright du road book. Une boîte de dialogue apparaît pour modifier le copyright.
