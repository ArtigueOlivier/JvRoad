JvRoad Manuel de référence

Les commandes du menu Livre

Nouveau   (, Alt+LN ou Ctrl+N)
Pour créer un nouveau road book.

Ouvrir   (, Alt+LO ou F3)
Pour ouvrir un road book existant. une boîte de dialogue pour sélectionner un fichier est affichée.Les anciens road book sont relus, en revanche, il est impossible de les enregistrer dans l'ancien format.

Fermer   (Alt+LF ou Ctrl+F4)
Pour ouvrir un road book existant. Les anciens road book sont relus, en revanche, il est impossible de les enregistrer dans l'ancien format.

Enregistrer   (, Alt+LE ou F2)
Pour enregistrer le road book en cours d'édition. Si le road book n'a jamais été enregistré, une boîte de dialogue pour saisir un nom de fichier est affichée. Les road books dans les anciens formats sont enregsitrés dans le format de JvRoad.

Enregistrer sous   (, Alt+LS ou Ctrl+F2)
Pour enregistrer le road book en cours d'édition dans un fichier en particulier. Une boîte de dialogue pour saisir un nom de fichier est affichée. Les road books dans les anciens formats sont enregsitrés dans le format de JvRoad.

Aperçu impression   ( ou Alt+LA)
Pour afficher la fenêtre d'aperçu avant impression. Cette fenêtre permet de modifier la présentation des pages et d'en visualiser le résultat sans qu'il soit nécessaire de les imprimer.

Configurer impression   (Alt+LM)
Pour afficher la boîte de dialogue de réglage des marges.

Imprimer   (, Alt+LI ou Ctrl+P)
Pour imprimer le road book en cours d'édition.

Imprimer légende   (Alt+LG)
Pour imprimer sur une feuille de papier la signification de chaque indication qui peuvent être utilisée lors de la rédaction d'un road book.

Imprimer feuille vierge   (Alt+LR)
Pour imprimer des feuilles vierges qui utilisent la présentation du road book en cours d'édition.

Quitter   (Alt+LA ou Alt+F4)
Pour quitter l'application. Si un road book a été modifié sans être enregistré, une confirmation est demandée avant de sortir.
