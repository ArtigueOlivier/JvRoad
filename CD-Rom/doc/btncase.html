JvRoad Manuel de référence

Les boutons concernant les données de la case

   Distance et indications   (Alt+DD ou Ctrl+K)
Pour éditer la distance et les indications. Une boîte de dialogue apparaît pour saisir les informations.

   Commentaire   (Alt+DC ou Ctrl+M)
Pour éditer le texte de la case. Une boîte de dialogue apparaît pour saisir les informations.

   Commentaire aligné à gauche
Pour aligner à gauche le texte de la case.

   Commentaire centré
Pour centrer le texte de la case.

   Rotation   (Ctrl+J)
Pour effectuer une rotation d'un quart de tour de la carte en cours d'édition. Cette fonctionnalité n'est pas disponible avec toutes les cartes, le bouton est affiché lorsque la carte active supporte la rotation.

   Edition texte   (Alt+DI ou Ctrl+Q)
Pour éditer le texte de la carte "Texte". Une boîte de dialogue apparaît pour éditer le texte. Ce bouton est visible uniquement quand la carte active est du type "Texte".

   Suppression texte   (Alt+DI ou Ctrl+Q)
Pour effacer le texte de la carte "Texte". Ce bouton est visible uniquement quand la carte active est du type "Texte".

   Texte(s) aligné(s) à gauche
Pour aligner à gauche le texte de la carte "Texte" ou "Lignes de texte".

   Texte(s) centré(s)
Pour centrer le(s) texte(s) de la carte "Texte" ou "Lignes de texte".
