JvRoad Manuel de référence

Les commandes du menu Présentation

Case - Défaut   (Alt+SCD)
Pour sélectionner la présentation par défaut pour le dessin des cases.

Case - Sans total   (Alt+SCS)
Pour sélectionner la présentation "Sans total" pour le dessin des cases.

Case - Rallye   (Alt+SCR)
Pour sélectionner la présentation "Rallye" pour le dessin des cases.

Case - Rallye orientée distance partielle   (Alt+SCP)
Pour sélectionner la présentation "Rallye orientée distance partielle" pour le dessin des cases.

Case - Rallye orientée distance totale   (Alt+SCT)
Pour sélectionner la présentation "Rallye orientée distance totale" pour le dessin des cases.

Case - Dessin maximum   (Alt+SCD)
Pour sélectionner la présentation "Dessin maximum" pour le dessin des cases.

Case - Dessin maximum orientée distance partielle   (Alt+SCX)
Pour sélectionner la présentation "Dessin maximum orientée distance partielle" pour le dessin des cases.

Case - Dessin maximum orientée distance totale   (Alt+SCO)
Pour sélectionner la présentation "Dessin maximum orientée distance totale" pour le dessin des cases.

Case - Moto   (Alt+SCM)
Pour sélectionner la présentation "Moto" pour le dessin des cases.

Case - Moto orientée distance partielle   (Alt+SCA)
Pour sélectionner la présentation "Moto orientée distance partielle" pour le dessin des cases.

Case - Moto orientée distance totale   (Alt+SCE)
Pour sélectionner la présentation "Moto orientée distance totale" pour le dessin des cases.

Case - Raid orientée distance partielle   (Alt+SCI)
Pour sélectionner la présentation "Raid orientée distance partielle" pour le dessin des cases.

Case - Raid orientée distance totale   (Alt+SCL)
Pour sélectionner la présentation "Raid orientée distance totale" pour le dessin des cases.

Page - Deux colonnes par page   ( ou Alt+SPD)
Pour indiquer que les cases sont dessinés sur deux colonnes à raison d'une page par feuille de papier.

Page - Une colonne par page   ( ou Alt+SPU)
Pour indiquer que les cases sont dessinés sur une seule colonne à raison d'une page par feuille de papier.

Page - Deux pages avec une colonne   ( ou Alt+SPP)
Pour indiquer que les cases sont dessinés sur une seule colonne à raison de deux pages par feuille de papier.

Page - Raid avec deux colonnes par page   ( ou Alt+SPR)
Pour indiquer que les cases sont dessinés sur deux colonnes avec une entête simplifiée à raison d'une page par feuille de papier.

Page - Raid avec une colonne par page   ( ou Alt+SPA)
Pour indiquer que les cases sont dessinés sur une seule colonne avec une entête simplifiée à raison d'une page par feuille de papier.

Page - Raid sur deux pages avec une colonne   ( ou Alt+SPI)
Pour indiquer que les cases sont dessinés sur une seule colonne avec une entête simplifiée à raison de deux pages par feuille de papier.

Options   (Alt+SO)
Pour afficher la boîte de dialogue qui permet affiner la présentation et l'organisation du road book (par exemple le nombre de cases par colonne ou le sens de lecture).

Filtres   (Alt+SF)
Pour afficher la boîte de dialogue qui permet de définir les filtres. Les filtres sont utilisés pour modifier la présentation et le dessin des cases sur une page en particulier.

Couleurs   (Alt+SL)
Pour afficher la boîte de dialogue qui permet de définir les couleurs pour le dessin des différents éléments du road book.
