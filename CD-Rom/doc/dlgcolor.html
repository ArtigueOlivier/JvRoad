La boîte de dialogue: Couleurs











Description générale

Cette boîte de dialogue permet de définir les différentes couleurs utilisées pour le dessin des éléments d'un road book au travers d'un ensemble de champs répartis sur cinq onglets.

L'onglet Fonds

Pour définir toutes les couleurs de fond utilisées lors du dessin d'un road book.

L'onglet Textes

Pour définir toutes les couleurs des différents textes constituant un road book.

L'onglet Traits

Pour définir toutes les couleurs des différents tracés des élément d'un road book.

L'onglet Dessins

Pour définir toutes les couleurs utilisées pour le dessin des indications ou certains éléments de certaines cartes.

L'onglet Clipart

Pour définir toutes les couleurs utilisées pour le dessin des clipart de la carte Clipart.
