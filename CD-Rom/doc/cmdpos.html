JvRoad Manuel de référence

Les commandes du menu Position

Première case   (, Alt+PP ou Ctrl+D)
Pour aller sur la première case.

Case Précédente   (, Alt+PC ou Ctrl+X)
Pour aller sur la case précédente.

Case suivante   (, Alt+PS ou Ctrl+U)
Pour aller sur la case suivante.

Dernière case   (, Alt+PD ou Ctrl+F)
Pour aller sur la dernière case.

Aller à   (Alt+PA ou Ctrl+G)
Pour aller sur une case. Une boîte de dialogue apparaît pour saisir le numéro de la case à atteindre.

Rechercher   (Alt+PR ou Ctrl+L)
Pour aller sur une case. Une boîte de dialogue apparaît pour afficher un résumé du road book afin de permettre de choisir le numéro de la case à atteindre.
