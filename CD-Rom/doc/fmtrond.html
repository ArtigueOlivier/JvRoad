Création d'un road book: La superposition de cartes

A présent, notre objectif est d'ajouter une nouvelle case à notre road book. Le dessin de cette nouvelle case (voir ci-dessous) est nettement plus complexe car il nécessite l'utilisation de trois cartes. Néanmoins cela reste facile à construire.



Nous n'attarderons pas les processus pour ajouter une case et définir sa distance partielle qui ont déjà été abordés. Pour réaliser ce dessin, nous allons utiliser les cartes Rond Point, Grille et Objets.

Première étape:
  		•  Choisir la carte Rond Point.
  		• Choisir le type de trait Chemin puis click dans le petit carré pour le placer.
  		• Choisir le type de trait Route puis click dans le petit carré pour le placer.
  		• Choisir le type de trait Route nationale puis click dans les deux petits carrés pour les placer.
    On obtient ainsi:



    Deuxième étape:
    Nous allons à présent utiliser une deuxième carte. Cette deuxième carte, nous allons la mettre sur la deuxième vue afin de ne pas perdre la première. Pour cela, choisir Deuxième carte dans le menu Cartes - Type de vue. Vous remarquer que notre dessin s'affiche en rouge au lieu d'en noir. C'est normal, en mode édition seul les traits de la carte courante apparaissent en noir, les autres sont affichés en rouge pour les différencier.



  		•  Choisir la carte Grille.
  		• Choisir le type de trait Chemin puis click dans les traits du quadrillage pour les placer.
  		• Choisir le type de trait Chemin - Origine puis click dans le trait du quadrillage pour le placer.
  		• Choisir le type de trait Chemin - Destination inverse puis click dans le trait du quadrillage pour le placer.
    On obtient ainsi:



    Troisième étape:
    Nous allons à présent utiliser la carte des objets. Cette carte possède sa propre vue. Pour cela, choisir Cartes des objets dans le menu Cartes - Type de vue.
  		•  Choisir l'objet Grande maison puis click dans l'un des carrés délimité par le quadrillage pour placer l'objet. Certains objets peuvent occuper la place de plusieurs carrés. Le click droit ou l'objet poubelle permettent de supprimer un objet.
    On obtient ainsi:



    Le choix de la vue Aperçue permet de visualiser le résultat final.
