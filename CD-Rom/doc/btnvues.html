JvRoad Manuel de référence

Les boutons concernant les vues

   Première carte   (Alt+CVP)
Pour sélectionner la première carte.

   Deuxième carte   (Alt+CVD)
Pour sélectionner la deuxième carte.

   Troisième carte   (Alt+CVT)
Pour sélectionner la troisième carte.

   Quatrième carte   (Alt+CVQ)
Pour sélectionner la quatrième carte.

   Carte des objets   (Alt+CVO)
Pour sélectionner la carte des objets.

   Carte d'aperçu   (Alt+CVA)
Pour sélectionner la carte de prévisualisation du résultat final de la case tel qu'elle sera imprimée.
